Using our To-Do list

Creating a new item in an empty list

To add a new item, we'll click on the add task (the "+") button at the bottom of the page. This will bring up the following textbox:

...where we will enter in the name of our task and then press enter.

Our new task has been successfully created!

Creating a new item in a non-empty list

Starting with our list from the last step, we'll click the add task button again.

We can now enter in the name of our new task.

Now we have two items in our list!

Marking an item completed

We can mark the item "Call Mom" completed simply by clicking on the checkbox next to the task name.

We've now marked the task as completed!

Renaming an item

We'll rename the task "Text John" to "Text John about bank statements" by clicking on the small "pencil" icon on the task.

Now we can enter in our new name for the task.

We've now renamed the task!

Showing only uncompleted items

We can show only uncompleted tasks by clicking on the "show" button in the top left corner.

Doing so allows us to only see tasks that have not been checked off by the user!

Deleting all completed items

Delete all completed tasks by clicking on the delete all button in the top right.

We've now completed the bulk action and deleted all the tasks marked as completed.
